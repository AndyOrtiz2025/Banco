# Simulación de Sistema de Atención en un Banco

Este programa simula un sistema de atención en un banco utilizando una cola (FIFO - First In, First Out). Permite a los usuarios agregar clientes, atenderlos en orden y mostrar la cola de espera. También se incluye una opción para marcar a los clientes como prioritarios.

## Funcionalidades

- **Agregar Cliente:** Permite agregar un nuevo cliente a la cola.
- **Atender Cliente:** Atiende al cliente que está al frente de la cola.
- **Mostrar Cola de Espera:** Muestra la lista de clientes en espera.
- **Clientes Prioritarios:** Permite dar prioridad a ciertos clientes (por ejemplo, personas mayores o con discapacidad).
